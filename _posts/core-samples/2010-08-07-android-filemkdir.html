今天在實作在 sdcard裡面建立一個檔案!

其實這沒什麼難度!

File file = new File("/sdcard/tmp/tmp.db");
file.createNewFile();


這樣乍看之下是沒甚麼問題

但如果沒有tmp此目錄的話!

則會出現exception

當然! 要建立一個目錄也很簡單!!只要用mkdir即可!

file.mkdir();


但是!

必須加入一個權限才可以建立目錄!

<uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE"></uses-permission>



但是建立檔案就不用了!
