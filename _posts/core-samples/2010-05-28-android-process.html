如果你按下返回按鍵!

或者按下HOME那顆按鈕!

雖然程式看似被關閉了!

但其實他的process還是存在的!

要怎麼徹底砍掉該process呢?

如下段CODE!




@Override
protected void onDestroy() {
     super.onDestroy();
     android.os.Process.killProcess(android.os.Process.myPid());
}





如果是在AVD裡面想要看是否真的有砍掉的話!

可以用ddms來觀看!

在你的sdk/tools/目錄底下!

打開後，可以按Device->Show process來觀看喔!


如果是在真實手機!

那就請去下載一些task的app吧!
